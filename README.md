# Customer-Churn-Analysis-and-Segmentation
End-to-End EDA Project using Python, Pandas and Seaborn
## Project Overview

This project analyzes customer churn behavior in a telecommunications company using Python. The objective is to identify key factors influencing customer churn, segment customers based on tenure, and generate business insights that can help improve customer retention and revenue.

## Objectives

* Perform data cleaning and preprocessing
* Conduct Exploratory Data Analysis (EDA)
* Analyze customer churn patterns
* Segment customers based on tenure
* Identify factors associated with customer retention
* Generate actionable business recommendations

## Dataset Information

The dataset contains customer demographic information, service details, billing information, contract details, payment methods, and churn status.

Key Features:

* Gender
* Senior Citizen Status
* Tenure
* Monthly Charges
* Total Charges
* Contract Type
* Payment Method
* Internet Service
* Churn Status

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

### 1. Data Cleaning

* Identified missing values
* Handled missing values in TotalCharges
* Checked for duplicate records
* Prepared the dataset for analysis

### 2. Exploratory Data Analysis (EDA)

* Distribution analysis
* Correlation analysis
* Churn distribution analysis
* Customer behavior analysis

### 3. Customer Segmentation

Customers were segmented into:

* 0–12 Months
* 13–36 Months
* 37+ Months

The segmentation helped analyze customer lifecycle stages and retention behavior.

### 4. Advanced Churn Analysis

Analyzed churn patterns across:

* Contract Types
* Payment Methods
* Customer Segments
* Senior Citizen Status
* Gender

## Key Findings

* Month-to-Month customers exhibited the highest churn rate.
* Electronic Check users showed significantly higher churn levels.
* Long-term customers generated higher revenue and demonstrated stronger retention.
* Senior citizens exhibited a comparatively higher churn rate than non-senior customers.
* Customer tenure showed a strong positive relationship with TotalCharges.

## Business Recommendations

* Focus retention efforts on new customers during their first year.
* Encourage customers to move to long-term contracts.
* Promote automatic payment methods to improve retention.
* Design targeted retention strategies for high-risk customer groups.
* Develop loyalty programs for long-term customers.

## Conclusion

The analysis identified customer tenure, contract type, and payment method as important factors associated with churn behavior. Retaining customers during the early stages of their lifecycle and encouraging long-term engagement can help improve customer retention and overall business performance.
